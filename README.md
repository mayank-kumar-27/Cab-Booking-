# Manish Tour and Travels

A modern, responsive website for cab booking and travel services in Purnia, Bihar.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Booking Form** - Easy-to-use form with WhatsApp integration
- **Car Fleet Showcase** - Display of available vehicles with background images
- **Popular Destinations** - Tourist places carousel with booking options
- **FAQ Section** - Accordion-style frequently asked questions
- **Smooth Animations** - Scroll reveal effects and smooth transitions
- **Touch-Friendly** - Drag-to-scroll carousels for mobile devices

## Tech Stack

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript (No frameworks)
- WebP images for optimized loading

## Configuration

To customize the website for your business:

1. **Change Phone/WhatsApp Number**:
   - Edit `script.js` line 2: `const WHATSAPP_NUMBER = '919934642587';`
   - Update phone links in `index.html` (search for `tel:+919934642587`)
   - Update WhatsApp links in `index.html` (search for `wa.me/919934642587`)

2. **Update Business Details**:
   - Company name, logo, and branding in `index.html`
   - Service areas and destinations
   - Operating hours in footer section

3. **Customize Images**:
   - Replace car images in `assets/` folder
   - Update destination images
   - Maintain WebP format for best performance
